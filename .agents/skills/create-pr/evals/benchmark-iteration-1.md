## 実行環境の注記

リモート実行環境(Claude Code on the web)で実施したため、以下の環境差分がある。

- `gh` CLI が無いため、スキル手順中の `gh pr create` / `gh auth setup-git` は GitHub MCP ツール(`create_pull_request` / `update_pull_request`)で代替した(eval 3, 4)。スキル本来の gh 経路は未検証。
- GitHub MCP サーバーが PR 本文末尾に `Generated by Claude Code` フッターを自動付加する。スキルが生成した本文自体には署名が無いことをもって PASS と判定した。
- 環境のプロキシポリシーによりリモートブランチ削除が拒否されるため、eval 3, 4 のクリーンアップでリモートブランチ `eval/create-pr-draft-test` / `eval/create-pr-template-test` が残存(PR #2, #3 はクローズ済み)。手元環境から `git push origin --delete <branch>` で削除する。
- eval 5 はリモートがローカル bare リポジトリのため PR 作成は必ず失敗する設計。判定対象は base 検出と失敗報告のみ。
